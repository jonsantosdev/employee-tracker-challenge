  # Employee Tracker

  ### Description
  This application allows the user to do the following with an employee database:
  * View all employees
  * View all departments
  * View all roles
  * Add an employee
  * Add a department
  * Add a role
  * Update an employee role

  ## Installation
  No installation required. It is a web application
